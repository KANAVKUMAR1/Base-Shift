# Base Shift – Utility

A lightweight web application for converting between binary, decimal, and text formats with a clean and responsive interface.

## Features

* Binary to Decimal conversion
* Decimal to Binary conversion
* Text to Binary conversion
* Binary to Text conversion
* One-click copy functionality
* Random sample generator
* Conversion explanation panel
* Responsive and modern user interface

## Live Demo

https://base-shift-omega.vercel.app/

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## Project Structure

```text
index.html
style.css
script.js
```

## Getting Started

1. Open the live website or clone the repository.
2. Select a conversion mode.
3. Enter the input.
4. View the converted output instantly.
5. Copy the result using the built-in copy button.

## Purpose

Base Shift was developed as a fast, browser-based utility for performing common binary, decimal, and text conversions while providing a simple and intuitive user experience.
